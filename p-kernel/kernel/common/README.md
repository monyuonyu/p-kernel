# T-Kernel 共通カーネルコード

このディレクトリには、T-Kernelのコア機能を実装する共通コードが含まれています。これらのコードはアーキテクチャに依存しない形で実装されており、すべてのプラットフォームで共有されます。

## 概要

T-Kernelは、リアルタイム組み込みシステム向けのマイクロカーネルオペレーティングシステムです。このcommonディレクトリには、プラットフォームに依存しないカーネルの中核機能が実装されています。

## 主要機能

### 1. タスク管理とスケジューリング
- **task.c**: タスクの状態管理、スケジューリング、実行可能キューの管理
- **task_manage.c**: タスクの生成・削除・状態変更
- **task_sync.c**: タスク間同期機能

### 2. メモリ管理
- **memory.c**: 基本メモリ管理機能
- **mempfix.c**: 固定長メモリプール管理
- **mempool.c**: 可変長メモリプール管理

### 3. 同期プリミティブ
- **semaphore.c**: セマフォ（資源の排他制御）
- **mutex.c**: ミューテックス（相互排除ロック）
- **eventflag.c**: イベントフラグ（イベント同期）
- **mailbox.c**: メールボックス（メッセージ交換）
- **messagebuf.c**: メッセージバッファ（可変長メッセージ）
- **rendezvous.c**: ランデブ（同期通信）

### 4. 高速ロック機能
- **fastlock.c**: 高速排他制御ロック
- **fastmlock.c**: 高速マルチロック
- **klock.c**: カーネルロック

### 5. デバイス管理
- **device.c**: デバイスドライバ登録・管理
- **deviceio.c**: デバイスI/O制御

### 6. 時間・タイマー管理
- **timer.c**: システムタイマー管理
- **time_calls.c**: 時間関連システムコール

### 7. システム管理
- **tkstart.c**: カーネル起動処理
- **subsystem.c**: サブシステム管理
- **objname.c**: オブジェクト名管理
- **wait.c**: 待ち状態管理

### 8. ユーティリティ
- **bitop.c**: ビット操作ライブラリ
- **misc_calls.c**: その他のシステムコール

### 9. T-Monitor機能
- **tm_command.c**: モニターコマンド処理
- **tm_exit.c**: モニター終了処理
- **tm_getchar.c**: 文字入力
- **tm_getline.c**: 行入力
- **tm_monitor.c**: モニターメイン
- **tm_printf.c**: フォーマット出力
- **tm_putchar.c**: 文字出力
- **tm_putstring.c**: 文字列出力

## アーキテクチャ特徴

### マイクロカーネル設計
T-Kernelはマイクロカーネル設計を採用しており、最小限の機能のみをカーネル空間で実行し、その他の機能はユーザー空間で実装されます。

### リアルタイム性
- 優先度ベースのプリエンプティブスケジューリング
- 割り込み処理の高速化
- 決定論的な応答時間

### ポータビリティ
- アーキテクチャ依存部分の分離
- 標準化されたAPI
- 設定可能なシステム構成

## ビルド設定

各機能は`USE_FUNC_*`マクロによって条件付きコンパイルされます。これにより、必要な機能のみを含むカーネルを構築できます。

## 開発者向け情報

### コーディング規約
- MISRA-C準拠
- 日本語ドキュメントコメント
- Doxygen形式のAPI文書

### デバッグサポート
`USE_DBGSPT`が有効な場合、各機能にデバッグサポート機能（`td_*`関数）が含まれます。

## x86/QEMU での動作確認

x86 ポート（`boot/x86/`）において以下の機能が実際に使用されています。

| 機能 | 利用箇所 |
|------|---------|
| `tk_cre_tsk` / `tk_sta_tsk` | シェルタスク・ネット RX タスク生成 |
| `tk_dly_tsk` | NIC ポーリング・DNS リトライ待機 |
| `tk_cre_sem` / `tk_wai_sem` / `tk_sig_sem` | TCP RX バッファのブロッキング受信 |
| `timer.c` | PIT（8254）割り込みで駆動するシステムクロック |
| `tkstart.c` → `usermain()` | ドライバ・ネットスタック初期化の起点 |
