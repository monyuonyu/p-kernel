# インクルードファイル

p-kernel プロジェクト全体で使用されるヘッダーファイルと API 定義です。

## サブディレクトリ

| ディレクトリ | 内容 |
|------------|------|
| `drivers/` | デバイスドライバ関連ヘッダー（H8300・RL78） |
| `kernel/` | T-Kernel コア API ヘッダー |
| `lib/` | ライブラリ関数ヘッダー |

> **x86 ポートのヘッダー** は `arch/x86/include/` に配置されています（`netstack.h`、`rtl8139.h`、`vga.h` 等）。
