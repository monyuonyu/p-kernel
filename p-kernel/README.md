# p-kernel

p-kernelは、RL78、H8300、x86などの複数のハードウェアアーキテクチャをサポートするリアルタイムカーネル実装です。このプロジェクトは、アーキテクチャ固有のコード、デバイスドライバ、システムライブラリを含む完全なカーネル環境を提供します。

## ディレクトリ構造

- `arch/` - サポートされているプラットフォーム向けのアーキテクチャ固有コード
- `boot/` - ブートローダーと初期化コード
- `docs/` - ドキュメントとマニュアル
- `drivers/` - 各種ハードウェアコンポーネント用のデバイスドライバ
- `include/` - ヘッダーファイルとAPI定義
- `kernel/` - コアカーネル実装
- `lib/` - サポートライブラリとユーティリティ
