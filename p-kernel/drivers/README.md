# デバイスドライバ

p-kernel がサポートする各ハードウェア向けのデバイスドライバです。

## サブディレクトリ

| ディレクトリ | 対象 | 状態 |
|------------|------|------|
| `common/` | アーキテクチャ共通ドライバコード | 実装済み |
| `h8300/` | ルネサス H8/300 向けドライバ | 実装済み |
| `rl78/` | ルネサス RL78 向けドライバ | 実装済み |

## x86/QEMU 向けドライバについて

x86 ポートのドライバは `arch/x86/` に実装されています。

| ドライバ | ファイル | 機能 |
|---------|---------|------|
| シリアル I/O | `arch/x86/sio.c` | COM1 (0x3F8) 送受信 |
| VGA テキスト | `arch/x86/vga.c` | 80×25 テキストモード、スクロール |
| PS/2 キーボード | `arch/x86/keyboard.c` | IRQ1、スキャンコード→ASCII |
| PCI バス | `arch/x86/pci.c` | コンフィグ空間列挙（0xCF8/0xCFC） |
| RTL8139 NIC | `arch/x86/rtl8139.c` | RX リングバッファ、TX 4 desc、IRQ11 |

詳細は [`arch/x86/README.md`](../arch/x86/README.md) を参照してください。
