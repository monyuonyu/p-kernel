# ブートコード

このディレクトリには、p-kernel の起動処理に関連するコードが含まれています。
各サブディレクトリは特定のハードウェアプラットフォーム向けの
ブートローダーと初期化コードを提供します。

## サブディレクトリ

| ディレクトリ | 対象プラットフォーム | 状態 |
|------------|-----------------|------|
| `x86/`    | QEMU x86_64     | ✅ **完全動作** — T-Kernel + TCP/IP スタック統合ビルド |
| `h8300/`  | ルネサス H8/300  | 実装済み |
| `rl78/`   | ルネサス RL78    | 実装済み |

## x86 ブート（主力）

`boot/x86/Makefile` が全ソース（boot + arch/x86 + kernel/common + lib）を
統合してシングルバイナリ `bootloader.bin` をビルドします。

```
Multiboot エントリ (start.S)
  → 32→64 ビットロングモード移行
  → IDT・PIC・タイマー初期化
  → T-Kernel 起動 (tkstart.c)
  → usermain() — シェル・NIC・ネットスタック起動
```

詳細は [`x86/README.md`](x86/README.md) を参照してください。
